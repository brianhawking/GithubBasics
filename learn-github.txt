Git Cheat Sheet



Setup 
Configuring user information used across all repos


• git config –-global user.name [name]
    -- Set a name that is identifiable for credit in revision history

• git config --global user.email [email address]
    -- set an email address that will be associated with each history



Setup & Init
Configuring user information, installing and closing repos


• git init
    -- Initializes the current directory as a git repo

• git clone [url]
    -- retrieve / clone an entire repo from online url

Stage & Snapshot
Working with snapshots and Git staging

• git status
    -- show modified or staged files in working directory

• git add
    -- Add a file as it looks now to your next commit

• git commit -m "message"
    -- commit your staged content




Branch & Merge
Isolating work in branches, changing context, and integrate changes



Share & Update
Configuring user information, installing and closing repos


• git remote add [alias] [url]
    -- add a git URL as an alias

• git fetch [alias]
    -- fetch down all the branches from that Git remote
